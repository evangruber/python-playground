# Python Playground 🚀

A structured sandbox for mastering Python fundamentals, experimenting with language features, automating tasks, and prototyping ideas.

## 📁 Repository Structure

* **🧪 Experiments/**: Quick, isolated tests and conceptual prototypes.
* **🤖 Automation/**: Practical, time-saving scripts and digital tools.
* **📚 Learning/**: Hands-on code snippets from tutorials, books, and documentation.

## 🛠️ Current Progress

- [x] `📚 Learning/slicing_playground.py` — Exploring list and string slicing syntax.
- [x] `📚 Learning/exceptions_playground.py` — Practicing exception handling with try/except and common built-in exceptions.
- [x] `📚 Learning/eoferror_playground.py` — Understanding `EOFError` and graceful handling of end-of-file input.
- [x] `📚 Learning/date_parsing_playground.py` — Parsing, formatting, and manipulating dates with the `datetime` module.
- [x] `📚 Learning/dictionaries_playground.py` — Exploring dictionary creation, access, updates, iteration, and common methods.
- [x] `📚 Learning/loops_playground.py` — Exploring `for` and `while` loops, `break`, `continue`, nested loops, and loop patterns.
- [x] `📚 Learning/cli_arguments_playground.py` — Learning how to read and validate command-line arguments using `sys.argv`.
